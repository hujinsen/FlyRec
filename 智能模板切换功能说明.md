# FlyRec 智能模板切换功能

## 功能概述

FlyRec 现在支持根据当前鼠标焦点所在的应用程序自动切换提示词模板，让语音识别结果更符合使用场景。

## 支持的应用类型和模板

### 🗣️ 聊天模板
**适用应用：**
- 微信 (wechat.exe)
- QQ (qq.exe, qqscm.exe)
- 钉钉 (dingtalk.exe)
- Microsoft Teams (teams.exe)
- Slack (slack.exe)
- Telegram (telegram.exe)
- Discord (discord.exe)

**模板特点：** 生成适合聊天的回复内容，语言风格轻松自然

### 📧 邮件模板
**适用应用：**
- Outlook (outlook.exe)
- Thunderbird (thunderbird.exe)
- Foxmail (foxmail.exe)
- 网易邮箱大师 (mailmaster.exe)

**模板特点：** 格式化为正式邮件，包含称呼、正文、结尾等完整结构

### 💻 代码模板
**适用应用：**
- Visual Studio Code (code.exe)
- Visual Studio (devenv.exe)
- PyCharm (pycharm64.exe)
- IntelliJ IDEA (idea64.exe)
- Notepad++ (notepad++.exe)
- Sublime Text (sublime_text.exe)
- Atom (atom.exe)
- WebStorm (webstorm64.exe)
- PHPStorm (phpstorm64.exe)
- CodeBuddy (codebuddy.exe)

**模板特点：** 生成代码实现或技术文档

### 📝 默认模板
**适用应用：**
- 记事本 (notepad.exe)
- 写字板 (wordpad.exe)
- Microsoft Word (winword.exe)
- Excel (excel.exe)
- PowerPoint (powerpnt.exe)

**模板特点：** 去除语气词，修正语法错误，保持原意

## 智能检测机制

### 1. 进程名检测
系统会检测当前活动窗口的进程名，如果匹配预设的应用列表，直接切换对应模板。

### 2. 窗口标题关键词检测
如果进程名未匹配，系统会分析窗口标题中的关键词：
- 聊天关键词：微信、wechat、qq、钉钉
- 邮件关键词：邮件、mail、outlook、邮箱
- 代码关键词：code、visual studio、pycharm、idea

### 3. 实时切换
每次开始语音识别时，系统会实时检测当前活动窗口并选择最合适的模板。

## 使用方法

### 启用智能模板切换
1. 打开 FlyRec 应用
2. 进入"设置"页面
3. 勾选"智能模板切换（根据当前应用自动选择模板）"选项

### 手动模板选择
如果需要手动指定模板，可以：
1. 取消勾选智能模板切换选项
2. 在"文本处理模板"中选择固定模板

## 测试方法

使用提供的测试脚本验证功能：
```bash
python test_smart_template.py
```

测试脚本会显示：
- 当前活动窗口的进程名
- 窗口标题
- 推荐的模板类型

## 示例场景

### 场景1：在微信中使用
- **检测结果**：聊天模板
- **语音输入**："今天天气不错，要不要出去走走"
- **输出**：生成轻松的聊天回复

### 场景2：在Outlook中使用
- **检测结果**：邮件模板
- **语音输入**："明天开会讨论项目进度"
- **输出**：格式化为正式邮件格式

### 场景3：在VS Code中使用
- **检测结果**：代码模板
- **语音输入**："写一个Python函数计算斐波那契数列"
- **输出**：生成完整的代码实现

### 场景4：在记事本中使用
- **检测结果**：默认模板

## 技术依赖

- `psutil`: 用于获取进程信息
- `pywin32`: 用于Windows窗口API调用

## 注意事项

1. 智能模板切换功能仅在Windows系统上可用
2. 需要安装psutil和pywin32依赖包
3. 如果依赖包未安装，系统会自动回退到手动模板选择模式
4. 可以随时在设置中开启或关闭智能模板切换功能

## 扩展性

可以通过修改`app_template_mapping`字典来添加更多应用程序的支持：

```python
self.app_template_mapping = {
    'your_app.exe': '模板类型',
    # 添加更多应用...
}
```

这个智能模板切换功能让FlyRec能够真正理解用户的使用场景，提供更精准、更实用的语音识别结果！
